# Trading Position Size Calculator

A simple web-based calculator to determine optimal position size based on risk, stop loss, and fees.

## Inputs
- Total Equity ($)
- Max Risk (%)
- Stop Loss (%)
- Trading Fee (%)

## How to Use
1. Enter your values.
2. Click **Calculate** to get the recommended position size.
